Beach Quest - Rate Trinidad and Tobago Beaches (In Development)

A responsive web application where users can rate and browse beaches from the islands of Trinidad and Tobago. Users can create, edit, and delete beaches uploaded to the site. The site was design with login, logout, gmaps integration, and comment functionality. Some technologies used for this web app are Nodejs, Mongodb, Mongolab, and Express js. Styling for the login, signup, and internal beach pages are currently in the works to be completed.
